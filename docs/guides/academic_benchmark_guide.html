üìö What is the "Academic Benchmark"?

The standard academic benchmark refers to running simulations with all 20 fault types using these parameters:

  • Duration: 8 hours (480 samples)
  • Fault start: 1 hour (60 samples normal, 420 samples with fault)
  • Used in: 100+ research papers since 1993
  • Purpose: Compare fault detection algorithms fairly

üéØ How to Run the Academic Benchmark

Standard Research Protocol: Most papers use faults 1-15 (skip 16-20 as they're unknown)
# Method 1: Run all known faults (1-15) from tep_simulator_easy import TEPSimulatorEasy simulator = TEPSimulatorEasy() # Standard academic benchmark for fault_type in range(1, 16): # Faults 1-15 print(f"Running Fault {fault_type}...") # Run fault scenario simulator.run_simulation( duration_hours=8, fault_type=fault_type, fault_start_hour=1, save_results=True, plot_results=False # Set True if you want plots ) # Run normal operation for comparison simulator.run_simulation( duration_hours=8, fault_type=0, save_results=True, plot_results=False ) print("Academic benchmark complete!")
# Method 2: Run specific fault categories # Easy faults (good for learning) easy_faults = [1, 2, 3, 4, 5, 6, 9] # Medium faults (intermediate) medium_faults = [7, 8, 10, 11, 12] # Hard faults (advanced research) hard_faults = [13, 14, 15] # Run easy faults first for fault in easy_faults: simulator.run_simulation(duration_hours=6, fault_type=fault, fault_start_hour=1)

üìä Understanding Your CSV Data

Each CSV file contains:

  • 41 XMEAS variables: Process measurements (temperatures, pressures, flows, compositions)
  • 11 XMV variables: Manipulated variables (valve positions, setpoints)
  • Time_Hours: Time column for easy plotting
  • Rows: One per sample (20 samples/hour, so 8 hours = 160 rows)
üí° Key Variables to Monitor:
  • XMEAS(7): Reactor pressure (kPa)
  • XMEAS(9): Reactor temperature (¬∞C) - Most important!
  • XMEAS(11): Product flow rate (m¬≥/h)
  • XMEAS(12): Reactor level (%)
  • XMEAS(13): Separator level (%)
